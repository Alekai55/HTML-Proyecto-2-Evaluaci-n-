# 🍀 ADROMI Tech - Consultoría de Desarrolladores

Bienvenido al repositorio oficial de **ADROMI Tech**. Somos un equipo de cuatro desarrolladores (estudiantes de DAM en Xtart) unidos para crear soluciones tecnológicas integrales.

Este espacio sirve como escaparate de los servicios que ofrecemos como consultoría. Como prueba tangible de nuestras capacidades, hemos desarrollado nuestro producto estrella: **Ecos del Bosque Roto**, un ecosistema completo que nos ha permitido unificar diseño web, programación de software, estructuración de datos y administración de sistemas en un único gran proyecto.

## 💼 Nuestros Servicios y Stack Tecnológico

Aplicamos de forma práctica nuestras áreas de especialización utilizando las siguientes herramientas:

* **Desarrollo Front-end y Diseño Web:** Creación de portales modernos y responsivos utilizando HTML5, Tailwind CSS y DaisyUI, con implementaciones dinámicas en JavaScript y animaciones AOS.
* **Desarrollo de Software y Lógica de Negocio:** Programación robusta y orientada a objetos en Java utilizando IntelliJ, incluyendo la creación de interfaces gráficas nativas con JPanel.
* **Arquitectura de Bases de Datos:** Diseño de modelos Entidad-Relación, normalización estructurada y consultas eficientes utilizando MySQL, MariaDB y DBeaver.
* **Sistemas Informáticos y Redes:** Instalación, configuración, administración y virtualización de entornos seguros utilizando Linux, Windows Server y VirtualBox.

## 📅 La evolución de nuestro producto estrella

De una idea en papel a un ecosistema funcional. Este ha sido el progreso de desarrollo de *Ecos del Bosque Roto*:

* **Septiembre 2025:** Primeros pasos con la estructura básica HTML y CSS de nuestra plataforma.
* **Octubre 2025:** Introducción a Java, estableciendo la Programación Orientada a Objetos del núcleo del juego.
* **Noviembre 2025:** Modelado y diagramas relacionales para la persistencia de datos.
* **Diciembre 2025:** Entrega de la primera versión funcional de *Ecos del Bosque* y despliegue de la web inicial.
* **Febrero 2026:** Segunda iteración y rediseño completo del front-end implementando Tailwind y DaisyUI.

## 🚀 Cómo visualizar la web de la consultoría

Este portal web no requiere procesos de *build* complejos ni instalación de dependencias pesadas, ya que hace uso de CDNs para optimizar la carga.

1. Clona este repositorio en tu equipo local o tu IDE preferido:
   ```bash
   git clone [https://github.com/Alekai55/HTML-Proyecto-2-Evaluaci-n-.git](https://github.com/Alekai55/HTML-Proyecto-2-Evaluaci-n-.git)
   ```
2. Navega hasta el directorio del proyecto.
3. Abre el archivo `index_tailwind_daisy.html` directamente en tu navegador web.
