Nuestro juego ha sido creado en Java utilizando el IDE IntelliJ que combinándolo con GitHub nos ha permitido trabajar de manera conjunta al crear interfaces gráficas mediante el uso de herramientas como JPanel

Para el diseño de nuestra Web hemos utilizado el IDE VistualStudioCode que junto con la herramienta Bootstrap aprendida en clase, nos ha permitido dar un diseño moderno y elegante a nuestro proyecto.

Mediante el uso de DBeaver con MariaDB hemos creado la base de datos implementando tablas acordes a la narrativa del juego y hemos puesto en práctica el uso de triggers, vistas y procedures.

Utilizando la herramienta Cisco Packet Tracer que nos permite simular redes informáticas, hemos creado y configurado por terminal una red LAN con swtiches y varios dispositivos.
